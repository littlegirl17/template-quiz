Trắc nghiệm HTML - CSS

123

Mô tả:

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu siêu văn bản được sử dụng để tạo ra các trang web. HTML cung cấp cấu trúc cơ bản của trang web, cho phép chúng ta định nghĩa các phần tử như tiêu đề, đoạn văn, hình ảnh, và liên kết. Mỗi trang web bạn thấy trên internet đều được xây dựng bằng HTML. CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách trình bày một tài liệu được viết bằng HTML. CSS xác định cách mà các phần tử HTML sẽ được hiển thị trên màn hình, bao gồm màu sắc, phông chữ, kích thước và bố cục. Với CSS, bạn có thể tạo ra các trang web đẹp mắt và thân thiện với người dùng.
